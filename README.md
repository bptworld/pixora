# Pixora Firmware Updates

Use the built-in Pixora server firmware update button to update a display.

## Update a Display

1. Open your Pixora server dashboard.
2. Select the display you want to update.
3. Click `Firmware Update`.
4. Choose the matching user OTA firmware file:
   - `pixora-v2.0.0-64x32-user-ota-firmware.bin`
   - `pixora-v2.0.0-128x32-user-ota-firmware.bin`
5. Start the update and leave the display powered on until it reboots.

Use the `user-ota` firmware for normal updates. It preserves the display's Wi-Fi and cloud setup.
