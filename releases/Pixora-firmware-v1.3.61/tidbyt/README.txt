Pixora firmware v1.3.61 - tidbyt

This is a generic setup-page firmware image.

Files:
- pixora-v1.3.61-tidbyt-ota-firmware.bin: OTA update after Pixora is already installed
- pixora-v1.3.61-tidbyt-usb-full-flash.bin: full USB flash image
- pixora-v1.3.61-tidbyt-bootloader.bin: advanced/manual flashing only
- pixora-v1.3.61-tidbyt-partition-table.bin: advanced/manual flashing only
- pixora-v1.3.61-tidbyt-flash-args.txt: advanced/manual flashing reference

No personal Wi-Fi, password, endpoint, or device name is baked into this build.

After a first USB flash, configure the device from the Pixora Windows app.
